Elm-Erlang Catalog
==================

Simple products catalog built with Erlang using ELM on the Frontend.

Requirements
------------

Required software/libraries:

* Erlang >= 19.0
* elm-lang >= 1.8


Run Locally
-----------

    $ make
    $ make run
    
    
Run in Docker
------------

    $ docker-compose build
    $ docker-compose up
    
Go to http://localhost:4000.
